How it works

  1. Point the camera so the shoe's label fills the frame — name, size box, and the price sticker.
  2. It reads the model name and the sticker price on the fly. Two ✓ chips show what it has.
  3. When it has both, it saves the box and you move to the next one. Can't read the price? Type it.
  4. Each saved box shows the discount and StockX / GOAT / eBay-sold links — the real "good deal?" answer.

Everything runs on your phone. Nothing is uploaded. Live camera needs an https:// page.
